# «Текстовый редактор»

### Памятка

#### 1. Скачать папку с проектом

#### 2. Установить зависимости
Установить зависимости можно командой:

```
npm install
```

#### 2. Собрать проект
Собрать для разарботки:

```
npm run dev
```

Запустить на локальном сервере:

```
npm run start
```

Собрать для продакшн:

```
npm run build
```

---

Собранный проект находится в папке /dist
